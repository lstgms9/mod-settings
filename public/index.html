Secure a box

      
Turn on SSH 2FA for any server using your existing authenticator code. Enter its IP and root password — we log in, set it up, and switch it on.

      

        
⚠ BEFORE YOU START

        
Open a PuTTY/SSH session to the box first and keep it open. If a 2FA login fails afterwards, run bash /root/2fa-revert.sh in that session (or the provider console) to undo it. Your authenticator code (the one you already use) becomes the box's login code too.
